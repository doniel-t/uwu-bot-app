# Self hosting guide

## requirements:
- node 16+

1. clone repo `git clone https://github.com/doniel-t/uwu-botv2.git`
2. install requirements `npm install`
3. `npm run build`
4. get static files from build directory

## developing:
1. clone repo `git clone https://github.com/doniel-t/uwu-botv2.git`
2. install requirements `npm install`
3. `npm run dev` -> runs on localhost:3000
